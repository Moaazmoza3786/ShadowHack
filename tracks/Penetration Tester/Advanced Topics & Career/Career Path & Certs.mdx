## Certification Tier List

1.  **OSCP (Offensive Security Certified Professional):** The HR Gold Standard. "Try Harder". 24-hour exam.
2.  **PNPT (Practical Network Penetration Tester):** More realistic. 5-day exam. Includes OSINT and Active Directory.
3.  **eJPT (Junior Penetration Tester):** Great entry-level practical exam.
4.  **CEH (Certified Ethical Hacker):** Good for HR, bad for skills (mostly multiple choice).
