# OSINT Frameworks

Open Source Intelligence (OSINT) is not just random Googling. 
It is structured data gathering.

## The OSINT Framework

A collection of tools and resources.
[osintframework.com](https://osintframework.com/)

## Tools

### 1. TheHarvester
Gathers emails, subdomains, hosts, employee names.

```bash
theHarvester -d target.com -b google,linkedin
```

### 2. Maltego
Visual link analysis. Connects people to domains, to emails, to social profiles.

### 3. Shodan
Google for devices. Find webcams, industrial control systems, and servers.

## Lab: Shodan Search

Query: `org:"BreachLabs" product:"Apache" port:8080`
