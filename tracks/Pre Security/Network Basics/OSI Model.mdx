# The OSI Model

Everything in networking—and by extension, hacking—is based on the OSI Model.
It breaks down the complex process of sending data into 7 manageable layers.

As a hacker, you need to know which layer an attack targets. Is it a physical wire tap (Layer 1)? Or a SQL Injection (Layer 7)?

## Layer Breakdown

### 7. Application (The Interface)
Where users interact.
*   **Protocols:** HTTP, SSH, DNS.
*   **Attacks:** SQL Injection, XSS, Phishing.

### 4. Transport (The Delivery)
Ensures data gets there (TCP) or just sends it fast (UDP).
*   **Protocols:** TCP, UDP.
*   **Attacks:** Port Scanning, SYN Floods.

### 3. Network (The Route)
Addressing and routing packets across the internet.
*   **Protocols:** IP, ICMP.
*   **Attacks:** IP Spoofing, Man-in-the-Middle (MitM).

### 2. Data Link (The Local)
Physical addressing on the local hardware.
*   **Protocols:** Ethernet, ARP, Wi-Fi.
*   **Attacks:** ARP Poisoning, MAC Flooding.
